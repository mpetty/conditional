# Conditional v0.1

Library for conditionally displaying html based on field values.

- Requires jQuery 1.7+

## Bower Installation

Add to your project's `bower.json` file, like:

```json
{
  "name": "my-project",
  "version": "1.0.0",
  "dependencies": {
    "jquery": "1.11.0",
    "modal": "git@github.com:mpetty/conditional"
  }
}
```

## Usage

```javascript
$('form').conditional(options);
```
